# User Stories

## Identity

### US-001
Jako návštěvník chci začít B2C guide bez okamžitého vytvoření plného profilu, abych nejprve získal hodnotu.

### US-002
Jako uživatel chci mít více rolí, protože mohu být současně klient, profesionál a člen firmy.

### US-003
Jako uživatel chci jasně vidět, v jakém kontextu právě jednám, abych omylem nepublikoval firemní obsah osobně.

## B2C Guide

### US-010
Jako klient chci popsat problém běžným jazykem, abych nemusel znát stavební terminologii.

### US-011
Jako klient chci u otázky odpovědět „nevím“, abych mohl pokračovat i bez odborných znalostí.

### US-012
Jako klient chci později pokračovat v nedokončeném guide.

### US-013
Jako klient chci vidět průběžné shrnutí, abych mohl opravit špatnou interpretaci.

### US-014
Jako klient chci, aby se další otázky měnily podle odpovědí.

### US-015
Jako klient s technickým problémem chci dostat bezpečnostní upozornění, pokud moje odpovědi naznačují potenciálně nebezpečný stav.

### US-016
Jako klient, který neví co potřebuje, chci dostat několik pravděpodobných cest řešení s vysvětlením.
