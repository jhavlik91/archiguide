## Prioritizační pravidlo
Dřívější package nesmí být rozšířen tak, že zablokuje pozdější multi-profession use cases.

## MVP
Packages 1–8 + základ Package 9 admin/moderation.

## Beta exit criteria
- guide completion stabilní,
- privacy incident critical = 0,
- messaging základ funkční,
- request-response flow dokončitelný,
- admin umí řešit reporty.

## Public launch exit criteria
- review abuse controls,
- verification workflow,
- notification preferences,
- monetization transparent,
- mobile core flows.
